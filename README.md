# customer-behavior-analysis-SQL-Python-PowerBI
End to end data analytics project using Python, PostgreSQL, and Power BI. Includes database loading, data cleaning, EDA, SQL-based analysis, interactive dashboard development, and business reporting with insights presented through a Power BI dashboard and Gamma-based presentation.
This project demonstrates an end-to-end data analytics workflow, covering data ingestion, exploratory analysis, SQL querying, dashboard development, and business reporting. The objective is to transform raw data into actionable insights using Python, PostgreSQL, and Power BI.
